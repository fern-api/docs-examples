title: Shared resource
subtitle: This page is shared across every version of the docs
slug: shared-resource
---

Content that doesn't change between releases — support links, legal notices, status page — can live in a single file and be referenced from every version's `navigation`.

This page is defined once in `fern/shared-pages/shared-resource.mdx` and pulled into both `latest.yml` and `v2.yml` via a relative path.
