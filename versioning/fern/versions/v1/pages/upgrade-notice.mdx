v1 reached end-of-life and will not receive further updates or security patches. All new integrations should target the [Latest](/welcome) version.

If you're currently on v1:

1. Skim the v2 [migration guide](/v2/migration) for the rename from `ExampleClient` to `Client`.
2. Then follow the Latest [quickstart](/quickstart) to pick up the new auth model.

There is no direct v1 → Latest migration guide; upgrade through v2.
