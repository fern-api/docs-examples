# Bien démarrer

Suivez ces étapes pour commencer rapidement.

## Étape 1 : Installer

```bash
npm install @fern-api/fern-cli -g
```

## Étape 2 : Initialiser

```bash
fern init
```

## Étape 3 : Générer

```bash
fern generate
```

C'est prêt ! Vérifiez le dossier de sortie pour voir les fichiers générés.
